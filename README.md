# food-101-dataset

[dataset link](https://www.kaggle.com/dansbecker/food-101)

## convert .h5 to TF.js

```shell
    sudo pip install tensorflowjs

    tensorflowjs_converter --input_format keras \
                        path/to/my_model.h5 \
                        path/to/tfjs_target_dir
```

## run file

`npm start`
